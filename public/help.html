Ajuda

Guia de primeira conexão e solução de problemas.

← Início Downloads

Primeira Conexão — App Desktop

1

URL do Repositório

No aplicativo, abra Configurações → Repositório Online.

URL Base:

https://datashow.meuibbv.com

Clique Salvar, depois Carregar Bibliotecas.

2

Token Bearer

Na mesma tela, preencha o campo Token Bearer com o token fornecido pelo administrador.

Cole apenas o token — não inclua a palavra Bearer.

Primeira Conexão — Planejamento Web

1

Acessar o Site

Acesse hinos.meuibbv.com. A tela de configuração abrirá automaticamente na primeira visita.

2

Preencher URL e Token

Informe a URL https://datashow.meuibbv.com e cole o token Bearer, depois clique Conectar.

Erros Comuns

"Sem bibliotecas carregadas"

O catálogo online está desativado ou o token está incorreto. Verifique URL e Token, clique Salvar e tente novamente.

"CLOUD_SYNC_DISABLED"

Ative Habilitar catálogo online nas configurações, confira URL/Token, clique Salvar e tente novamente.

"AUTH_REQUIRED"

O token não confere com o do servidor. Verifique se copiou o token completo sem espaços extras.

Verificar Conexão no Terminal

curl -H "Authorization: Bearer SEU_TOKEN" \
  https://datashow.meuibbv.com/api/repository/libraries

Se retornar {"ok":true,...} a conexão e o token estão corretos.
